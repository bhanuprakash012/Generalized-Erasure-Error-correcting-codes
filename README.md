# Generalized erasure error-correcting stabilizer code utilizing kicked Ising dynamics.

## Overview
This repository contains a Jupyter notebook demonstrating the analysis of an $[[n,1,\frac{n}{2}-1^*]]$ generalized erasure error-correcting stabilizer code. 

## Key Features
* **Kicked Ising Dynamics:** Utilizes specific Hamiltonian dynamics to construct the code.
* **Erasure Correction:** Designed to correct all Pauli errors localized within a known erasure region of $\frac{n}{2}-1$ consecutive qubits.
* **Classical Simulation:** The notebook contains the simulations to verify the syndrome uniqueness in turn verifies the code.

## Packages and Languages Used
* Python
* Qiskit, Numpy.
